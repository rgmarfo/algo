EXAMPLE

Find the root of the equation f(x) = x3 - 2x2 + 3x - 5 = 0 using the False Position method. Start with the initial interval [1, 2] and perform three iterations.


SOLUTION

Equation:

f(x) = x3 - 2x2 + 3x - 5


Initial Interval:

[1, 2]


  • f(1) = 13 - 2(1)2 + 3(1) - 5 = -3
  • f(2) = 23 - 2(2)2 + 3(2) - 5 = 1

Since f(1) and f(2) have opposite signs, there is at least one root in the interval [1, 2].


Iteration 1:

  • Calculate the point c1

    c1 = 1.75

  • Evaluate f(c1):

    f(1.75) = -0.515625

  • Since f(1) and f(1.75) have opposite signs, update the interval to [1.75, 2].

Iteration 2:

  • Calculate the point c2

    c2 ≈ 1.834

  • Evaluate f(c2):

    f(1.834) ≈ -0.093

  • Since f(1.75) and f(1.834) have opposite signs, update the interval to [1.834, 2].

Iteration 3:

  • Calculate the point c3

    c3 ≈ 1.847

  • Evaluate f(c3):

    f(1.847) ≈ -0.011


Conclusion:

After three iterations, the approximate root of the equation f(x) = x3 - 2x2 + 3x - 5 = 0 is c3 ≈ 1.847.
