EXERCISE

1. Find the root of the equation \( f(x) = x^3 - 4x + 1 = 0 \) using the False Position method. Start with the initial interval \([0, 1]\) and perform three iterations.


2. Apply the False Position method to find the root of the equation \( f(x) = \cos(x) - x = 0 \). Use the initial interval \([0, \frac{\pi}{2}]\) and perform four iterations. What is the approximate root after the fourth iteration?


3. Consider the function \( f(x) = x^2 - 2x - 3 \). Use the False Position method with the interval \([1, 4]\) to find the root. Discuss the convergence of the method for this specific function and interval.


4. Use the False Position method to find the root of the equation \( f(x) = e^x - 3x \) within the interval \([0, 1]\). Perform three iterations and calculate the approximate value of the root.


5. Given the function \( f(x) = x^3 - 2x^2 - 5 \), apply the False Position method with the interval \([2, 3]\) and perform two iterations. Estimate the error \( |x_{true} - x_{approx}| \) after the second iteration, assuming the exact root lies within the interval.
