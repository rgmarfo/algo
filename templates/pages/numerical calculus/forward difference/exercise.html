EXERCISE


  1. 1. Use the Bisection method to find \( p_3 \) for \( f(x) = \sqrt{x} - \cos x \) on \([0, 1]\).

  2. 2. Let \( f(x) = 3(x + 1)\left(x - \frac{1}{2}\right)(x - 1) \). Use the Bisection method on the following intervals to find \( p_3 \).
    • (a) \([-2, 1.5]\)
    • (b) \([-1.25, 2.5]\)
  3. 3. Use the Bisection method to find solutions accurate to within \( 10^{-2} \) for \( x^3 - 7x^2 + 14x - 6 = 0 \) on each interval.
    • (a) \([0, 1]\)
    • (b) \([1, 3.2]\)
    • (c) \([3.2, 4]\)

  4. 4.

    Use the Bisection method to find solutions accurate to within \( 10^{-2} \) for \( x^4 - 2x^3 - 4x^2 + 4x + 4 = 0 \) on each interval.
    • (a) \([-2, -1]\)
    • (b) \([0, 2]\)
    • (c) \([2, 3]\)
    • (d) \([-1, 0]\)

  5. 5

    Use the Bisection method to find solutions accurate to within \( 10^{-5} \) for the following problems.
    • (a) \( x - 2^{-x} = 0 \) for \( 0 \leq x \leq 1 \)
    • (b) \( e^x - x^2 + 3x - 2 = 0 \) for \( 0 \leq x \leq 1 \)
    • (c) \( 2x \cos(2x) - (x + 1)^2 = 0 \) for \( -3 \leq x \leq -2 \) and \( -1 \leq x \leq 0 \)
    • (d) \( x \cos x - 2x^2 + 3x - 1 = 0 \) for \( 0.2 \leq x \leq 0.3 \) and \( 1.2 \leq x \leq 1.3 \)

  6. 6

    Use the Bisection method to find solutions accurate to within \( 10^{-5} \) for the following problems.
    • (a) \( 3x - e^x = 0 \) for \( 1 \leq x \leq 2 \)
    • (b) \( 2x + 3 \cos x - e^x = 0 \) for \( 0 \leq x \leq 1 \)
    • (c) \( x^2 - 4x + 4 - \ln x = 0 \) for \( 1 \leq x \leq 2 \) and \( 2 \leq x \leq 4 \)
    • (d) \( x + 1 - 2 \sin (\pi x) = 0 \) for \( 0 \leq x \leq 0.5 \) and \( 0.5 \leq x \leq 1 \)
