EXERCISE


1. Solve the following system of linear equations using Gaussian elimination: \[ \begin{aligned} 3x + 2y - z &= 1 \\ 2x - 2y + 4z &= -2 \\ -x + \frac{1}{2}y - z &= 0 \end{aligned} \] 2. Solve the following system of linear equations using Gaussian elimination: \[ \begin{aligned} 4x - 2y + z &= 6 \\ 2x + y - 3z &= 1 \\ -x + \frac{1}{2}y + 2z &= 3 \end{aligned} \] 3. Solve the following system of linear equations using Gaussian elimination: \[ \begin{aligned} 5x + 3y - 2z &= 7 \\ 2x - y + 4z &= 5 \\ 3x + 4y + z &= -2 \end{aligned} \] 4. Solve the following system of linear equations using Gaussian elimination: \[ \begin{aligned} x - 3y + 2z &= 4 \\ 2x + y - z &= -1 \\ -x + 2y + 3z &= 7 \end{aligned} \] 5. Solve the following system of linear equations using Gaussian elimination: \[ \begin{aligned} 3x + 4y - 2z &= 5 \\ 2x - y + z &= 1 \\ -x + 2y + 3z &= 4 \end{aligned} \]
