LECTURE NOTE


The Jacobi method iteratively refines an initial guess to the solution of the system. The matrix \( A \) is decomposed into three components:

\[ A = D + L + U \]

where:

  • \( D \) is the diagonal matrix containing the diagonal elements of \( A \),
  • \( L \) is the lower triangular matrix with elements below the diagonal (without the diagonal),
  • \( U \) is the upper triangular matrix with elements above the diagonal (without the diagonal).

The system \( Ax = b \) can then be rewritten as:

\[ Dx = b - (L + U)x \]

The Jacobi method isolates the diagonal component to iterate towards the solution:

\[ x^{(k+1)} = D^{-1}\left(b - (L + U)x^{(k)}\right) \]

or more explicitly, for each element \( i \):

\[ x_i^{(k+1)} = \frac{1}{a_{ii}} \left(b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right) \]

where \( x_i^{(k+1)} \) is the value of the \( i \)-th variable in the next iteration, and \( x_j^{(k)} \) are the values of the other variables from the current iteration.


Iterative Process

  1. Initial Guess: Start with an initial guess \( x^{(0)} \).
  2. Update Step: For each \( i \), compute the next approximation \( x_i^{(k+1)} \) using the current values \( x_j^{(k)} \).
  3. Convergence Check: After each iteration, check whether the difference between the new and old approximations is below a pre-defined tolerance level. If not, proceed to the next iteration.
  4. Repeat until the solution converges, i.e., when \( \| x^{(k+1)} - x^{(k)} \| \) is sufficiently small.

Convergence Criteria

The convergence of the Jacobi method is guaranteed if the matrix \( A \) is either diagonally dominant or symmetric and positive definite. A matrix \( A \) is said to be diagonally dominant if for each row \( i \):

\[ |a_{ii}| > \sum_{j \neq i} |a_{ij}| \]

If \( A \) is not diagonally dominant, the method may not converge, or it may converge very slowly.


ADVANTAGES

  • 1. Simple to implement and understand.
  • 2. Suitable for parallel computation since each component of \( x^{(k+1)} \) can be updated independently.
  • 3. Useful for large, sparse matrices where direct methods are inefficient.

DISADVANTAGES

  • 1. Convergence can be slow, especially if the system is not well-conditioned.
  • 2. Requires more iterations compared to more sophisticated methods like the Gauss-Seidel or Successive Over-Relaxation (SOR) methods.
  • 3. Only applicable to certain types of matrices (e.g., diagonally dominant).
