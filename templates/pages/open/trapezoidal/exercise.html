EXERCISE


1. Use the Trapezoidal Rule with \( n = 4 \) to approximate the integral \(\int_{0}^{2} x^2 \, dx\).


2. Approximate the integral \(\int_{1}^{3} \ln(x) \, dx\) using the Trapezoidal Rule with \( n = 6 \).


3. Apply the Trapezoidal Rule with \( n = 8 \) to estimate \(\int_{0}^{\pi} \sin(x) \, dx\).


4. Approximate the integral \(\int_{1}^{2} e^{x^2} \, dx\) using the Trapezoidal Rule with \( n = 5 \).


5. Use the Trapezoidal Rule with \( n = 4 \) to approximate the integral \(\int_{0}^{2} |x - 1| \, dx\).
