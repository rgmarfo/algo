EXERCISE


1. Find the fixed point of the function \( g(x) = \cos(x) \) starting with an initial guess \( x_0 = 0.5 \). Use the fixed-point iteration formula and perform at least five iterations.


2. Consider the function \( g(x) = \sqrt{1 + x} \). Determine whether this function has a fixed point in the interval \([0, 1]\). If so, use the fixed-point iteration method to approximate the fixed point, starting with an initial guess \( x_0 = 0.2 \).


3. Solve the equation \( x^2 - 3x + 2 = 0 \) using the fixed-point method. Rewrite the equation in the form \( x = g(x) \), choose an appropriate initial guess, and perform at least four iterations.


4. Use the fixed-point iteration method to find the fixed point of the function \( g(x) = 2 - \frac{1}{x} \) starting from \( x_0 = 1.5 \). Perform at least five iterations and check the convergence.


5. Determine if the function \( g(x) = \frac{1}{3} (2x + 5) \) has a fixed point. If it does, use the fixed-point method starting with an initial guess \( x_0 = 2 \). Perform at least four iterations to approximate the fixed point.
