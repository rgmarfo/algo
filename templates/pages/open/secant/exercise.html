EXERCISE

1. Find a root of the equation \(f(x) = x^3 - 6x^2 + 11x - 6\) using the Secant Method. Start with the initial guess \( x_0 = 2 \).



2. Use the Secant Method to find a root of the function \( f(x) = \sin(x) - \frac{x}{2}\) beginning with the initial guess \( x_0 = 1 \).



3. Apply the Secant Method to approximate a root of the equation \(f(x) = x^4 - 5x^2 + 4\) starting with the initial guess \( x_0 = 1 \).




4. Find a root of the function \( f(x) = \tan(x) - x \) using the Secant Method. Begin with the initial guess \( x_0 = 0.5 \).



5. Use the Secant Method to solve for a root of the equation \( f(x) = x^2 - \sqrt{x + 1} \) starting with the initial guess \( x_0 = 2 \).
